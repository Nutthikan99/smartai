# SmartKey FastAPI backend

## Quick start

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
cp .env.example .env
uvicorn main:app --reload --host 0.0.0.0 --port 8000
```

Open:
- API docs: `http://127.0.0.1:8000/docs`
- Health: `http://127.0.0.1:8000/health`

## Endpoints
- `POST /api/create-payment-session`
- `GET /api/payment-status/{transaction_id}`
- `POST /api/mock-pay/{transaction_id}`
- `POST /webhooks/promptpay`
- `POST /webhooks/stripe`

## Sample create request

```json
{
  "tag_id": "NFC001",
  "provider": "promptpay",
  "amount": 50
}
```

## Notes
- PIN is random 4 digits and generated by backend only.
- Transactions are stored in SQLite (`smartkey.db`).
- PromptPay QR payload in this template is a placeholder. Replace `build_promptpay_qr_payload()` with your real PSP/bank payload generator.
- Stripe checkout session is created only if `STRIPE_SECRET_KEY` is configured.
