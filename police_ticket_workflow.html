Police Ticket Workflow

ระบบบันทึกใบสั่งสำหรับเจ้าหน้าที่ตำรวจ

หน้านี้ถูกจัดใหม่เป็น workflow 3 ขั้นตอนชัดเจน: ถ่ายรูป → ตรวจสอบข้อมูลใบสั่ง → สร้าง PIN / ลิงก์ / เขียน NFC

1

ถ่ายรูปรถและอ่านป้ายทะเบียน

เปิดกล้องจากหน้าเว็บ กด Capture แล้วใช้ OCR ช่วยกรอกเลขทะเบียนอัตโนมัติ

เมื่อจัดมุมกล้องเรียบร้อยแล้ว ให้กด Capture รูป
ยังไม่มีรูป
สถานะกล้อง: ยังไม่เปิด
สถานะ OCR: ยังไม่เริ่ม
แนะนำให้ใช้มือถือหรืออุปกรณ์ที่มีกล้อง และควรเปิดผ่าน HTTPS เพื่อให้เบราว์เซอร์อนุญาตเข้าถึงกล้อง
2

ตรวจสอบและกรอกข้อมูลใบสั่ง

เลขที่ใบสั่งและวันเวลาจะรันอัตโนมัติ ส่วนสถานที่ ฐานความผิด และยอดค่าปรับกรอกได้ทันที

สถานะฟอร์ม: รอกรอกข้อมูล
3

ดำเนินการต่อแบบลำดับงานจริง

จัดใหม่ให้เป็นชุดคำสั่งตาม requirement แบบอ่านแล้วเข้าใจทันที

3.1
สร้าง PIN ปลดล็อค
ระบบจะสร้าง PIN 3 หลักอัตโนมัติ พร้อมเตรียม Link สำหรับเจ้าของรถ
3.2
บันทึกข้อมูลและรับ Link
บันทึกข้อมูลใบสั่ง แล้วสร้าง URL สำหรับส่งให้เจ้าของรถใช้ชำระเงิน
3.3
วางแผ่น NFC
นำแผ่น NFC มาแตะใกล้อุปกรณ์ในกรอบด้านล่าง เพื่อเตรียมเขียน Link ลงแผ่น
📶
จุดวางแผ่น NFC

วางหรือแตะการ์ด/แท็ก NFC ไว้ใกล้โทรศัพท์หรืออุปกรณ์ที่รองรับ Web NFC แล้วกดปุ่มเขียน Link ในข้อ 11

วางแผ่น NFC ไว้ในจุดนี้
ระบบจะเขียน URL ลง NFC ผ่าน Web NFC เมื่อกดปุ่มข้อ 3.4
3.4
เขียน Link ลง NFC
เมื่อแผ่น NFC พร้อมแล้ว ระบบจะเขียน URL ลงแผ่นเพื่อให้เจ้าของรถสแกนใช้งานได้
3.5
แจ้งสถานะการเขียนข้อมูล
ระบบจะแจ้งผลว่าบันทึกข้อมูลลง NFC ได้เรียบร้อยหรือไม่
PIN 3 หลัก
---
Public URL
ยังไม่ได้สร้างลิงก์
NFC Status
ยังไม่ได้เขียน NFC

-

-

-
